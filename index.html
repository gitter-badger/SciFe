SciFe

Scala Framework for Efficient Generation of Data Structures with Invariants

Download this project as a .zip file Download this project as a tar.gz file

SciFe is a tool for automated generation of complex structures, suitable for tasks such as automated testing and synthesis. SciFe is capable of exhaustive, memoized enumeration of values from finite or infinite domains. SciFe is introduces concepts of enumerators and enumerator combinators. SciFe is a Scala library that implements a domain-specific language. SciFe can generate complex structures such as search trees and models of class hierarchies. SciFe demonstrates better performance and shorter specifications when compared to existing approaches.
